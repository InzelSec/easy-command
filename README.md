Easy Command is a web-based tool designed to simplify the generation and copying of common commands used in **ethical penetration testing** and **security labs**.

The project aims to reduce friction during learning, labs, CTFs, and authorized assessments by providing ready-to-use command templates in a clean and accessible interface — similar in spirit to tools like revshell generators, but extended to other common utilities (e.g. enumeration tools).

---

## Features

- One-click copy for commonly used pentest commands
- Supports multiple tools and use cases (e.g. enumeration, networking, shells)
- Designed for learning, labs, and authorized security testing
